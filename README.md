## Welcome 👋

Welcome to the Splay Discord Bot! The Splay Discord Bot is a versatile and customizable bot designed to enhance your Discord servers. With a wide range of features and commands, this bot provides a fun and useful experience for your community.

## Description 📚

The Splay Discord Bot is a powerful and user-friendly bot developed to add functionality to your server and improve your community's Discord experience. With a wide range of features, including slash commands and simple music commands, Splay is the perfect addition to any Discord server.

## **Features**
- 🤖 **Versatility:** The bot offers a wide range of features, from entertainment to moderation, to provide diverse capabilities to your server.

- 📈 **Dashboard:** Conveniently manage the bot through a user-friendly dashboard. Customize settings, monitor activities, and maintain control.

- 🖼️ **Image Captioning:** Personalize your images by adding text captions. Add a personal touch to your graphics.

- 🛡️ **Moderation:** Maintain control of your server and create a pleasant atmosphere with integrated moderation commands. Protection and order come first.

- 💌 **Invites System:** Keep track of members who invite new members to your server. A useful tool to encourage participation.

- And much more... The possibilities are endless, allowing you to further expand the bot and make your server truly unique.

## To-Do 📝
- [ ] Don't want to host the bot yourself?

Before you can use Splay, make sure you meet the following requirements:

- Install [Visual Studio Code](https://code.visualstudio.com/download)
- [Node.js](https://nodejs.org/en/download/current) v16 or higher
- **Discord.js Library:** The bot's functionality is based on the Discord.js library. Install it through Node.js with the command: `npm install discord.js@latest`.
- You need a Discord bot token. Create your own bot on the [Discord Developer website](https://discord.com/developers/applications) and copy the token.
- [MongoDB](https://www.mongodb.com/try/download/community) (Required)

## Installation Guide 🏁

Follow these steps to install Splay on your server:

1. [Fork the Repository](https://github.com/XSaitoKungX/Splay/fork)
2. Clone your fork: `git clone https://github.com/your-username/Splay.git`
3. Create a feature branch: `git checkout -b my-new-feature`
4. Stage your changes: `git add .`
5. Commit your changes: Use `cz` or `npm run commit`, but not `git commit`
6. Push your branch: `git push origin my-new-feature`
7. Create a pull request

## 🤖 Bot Configuration
```yaml
# Please change the following values according to your requirements:

# Rename .env.example to .env and fill in the fields

apiKey=Your_API_Token  # Example: https://platform.openai.com/account/api-keys
WEBHOOKURL=Your_Guild_Create_Delete_Logs_Webhook_URL
MONGO_CONNECTION=Your_Mongo_DB_URL
BOTPREFIX=s+

# Configure your bot settings in config/example.json
change it to config/config.json:
{
    "name": "Splay",
    "token": "YOUR_DISCORD_BOT_TOKEN",  # Get this token here: https://discord.com/developers/applications/
```
